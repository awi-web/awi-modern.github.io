# AWI Modern Customer App

Diese Version wurde als moderne Single-Page-App umgesetzt.

## Funktionen

- Kunden-Identifikation mit Registrierung und Login (lokal via `localStorage`).
- Persönliches Dashboard mit Projektzahlen.
- Erstellung und Verwaltung von Webseiten-Projekten pro Unternehmen.
- Zentraler Bereich mit allen Demo-Webseiten (Vorlagen).
- Erfassung und Nachverfolgung von Kundenwünschen.

## Hinweis

Aktuell ist es eine Frontend-Demo ohne Backend/API. Für produktive Nutzung sollten Authentifizierung, Datenhaltung, Rollen und Sicherheit serverseitig umgesetzt werden.
